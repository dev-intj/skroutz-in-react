Started this project to improve skills on React and Redux,

## Development setup

Install packages for this repo. ( I recommend yarn )

```sh
yarn install
```

(or)

```sh
npm install
```

## Run the website locally

Running this project locally for devs.

```sh
yarn dev
```

(or)

```sh
npm run dev
```
